# Smart IT Help Desk System

A web-based IT Help Desk System developed using PHP and MySQL to streamline IT support requests and ticket management.

## Features

- Secure Login
- Admin Dashboard
- Ticket Statistics
- Search Tickets
- View Ticket Details
- Update Ticket Status
- Delete Tickets
- Responsive Design

## Technologies

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap Icons
